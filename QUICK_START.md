## üìÅ Key Files to Customize

1. **Contact Information**: 
   - `components/Footer.tsx` (lines with phone, email, address)
   - `app/contact/page.tsx` (contact details section)

2. **Company Story**: 
   - `app/about/page.tsx` (Our Story section)

3. **Services Details**: 
   - `app/services/page.tsx` (service descriptions and features)

4. **Images**: 
   - `app/gallery/page.tsx` (replace placeholders)
   - See `public/ASSETS_README.md` for detailed instructions

5. **Videos**: 
   - `components/HeroSection.tsx` (homepage banner)
   - `app/services/page.tsx` (services video)
